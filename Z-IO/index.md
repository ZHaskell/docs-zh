# Z-IO

{::comment}
Z-IO package provides high-performance I/O operations based on libuv's event loop and GHC lightweight threads.
{:/}

Z-IO 基于 [libuv](https://libuv.org/) 的事件循环和 GHC 的轻量线程提供了一些高性能的 I/O 操作。
